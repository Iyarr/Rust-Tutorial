## 使用するコマンド

ファイルのコンパイル

> rustrc ./(ファイル名)

cargo のコマンド

> プロジェクト作成
>
> > cargo new (project 名)
> > cargo build(target/debug/フォルダにファイルが作成される)
> > cargo run (ビルドと実行)
> > cargo check
